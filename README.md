# Portfolio Website (Scaffold)

This project is a simple scaffold for a personal portfolio website with three pages: Home, About, and Contact. Page contents will be added later.

Files:
- `index.html` — Home
- `about.html` — About
- `contact.html` — Contact
- `css/style.css` — Shared styles
- `js/main.js` — Minimal JS

To view locally, open `index.html` in a browser or use a static server such as `npx http-server`.
